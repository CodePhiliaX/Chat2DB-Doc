### Additional Code Examples

1. **Counting Employees by Department**  
   To count the number of employees in each department, you can ask:

   ```
   How many employees are in each department?
   ```

   And the generated SQL query would be:

   ```sql
   SELECT department, COUNT(*) AS employee_count FROM employees GROUP BY department;
   ```

2. **Finding the Average Salary**  
   To find the average salary of employees, you might type:

   ```
   What is the average salary of employees?
   ```

   The resulting SQL query would be:

   ```sql
   SELECT AVG(salary) AS average_salary FROM employees;
   ```
