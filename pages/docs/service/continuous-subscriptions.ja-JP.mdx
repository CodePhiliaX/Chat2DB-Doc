# 連続サブスクリプションサービス契約

**CHATDATA PTE. LTD. 連続サブスクリプションサービス契約**

**バージョン日：2024年6月**

本連続サブスクリプションサービス契約（以下「本契約」）は、CHATDATA PTE. LTD.（以下「本会社」または「私たち」）と、Chat2DB Pro製品の連続サブスクリプションサービスに関して締結されるものです。サービスを利用する前に、本契約の全ての条項をよくお読みいただき、十分にご理解ください。本契約のいずれかの条項に同意しない場合は、サービスを利用しないでください。サブスクリプションを選択し、サービスを利用することにより、本契約の全ての条項に同意したものとみなされます。

**1. 定義**

1.1 Chat2DB Pro：本会社が開発および運営する汎用データベース管理ツールを指します。

1.2 連続サブスクリプションサービス：ユーザーがサブスクリプション料金を支払うことにより、月次または四半期ごとにChat2DB Proが提供する関連サービスを利用するサービスを指します。

**2. サービス内容**

2.1 本会社は、ユーザーにChat2DB Proの使用権限および関連する技術サポートサービスを提供します。

2.2 ユーザーは、異なるサブスクリプションプランに応じて異なる機能およびサービスを享受できます。具体的な内容は、ユーザーが選択したサブスクリプションプランに基づきます。

**3. サブスクリプションおよび料金**

3.1 ユーザーは、Chat2DB Proサービスを月次または四半期ごとにサブスクリプションすることができます。サブスクリプション料金は、各サブスクリプション期間の開始時に、ユーザーが提供した支払い方法から自動的に差し引かれます。

3.2 サブスクリプション料金は、本会社の公式ウェブサイトまたは関連プラットフォームに掲載されている価格に基づきます。本会社は、市場状況に応じてサブスクリプション料金を調整する権利を有し、調整前に合理的な方法でユーザーに通知します。

3.3 ユーザーがサブスクリプション期間内にサブスクリプションをキャンセルした場合、支払済みのサブスクリプション料金は返金されず、サービスは現在のサブスクリプション期間の終了まで継続されます。

**4. 自動更新**

4.1 サービスの継続性を確保するために、ユーザーのサブスクリプションは各サブスクリプション期間の終了時に自動的に更新され、対応する料金がユーザーが提供した支払い方法から差し引かれます。

4.2 ユーザーは、アカウント設定を通じていつでも自動更新をキャンセルすることができます。キャンセル後、サービスは現在のサブスクリプション期間の終了まで継続され、次の期間は自動的に更新されません。
