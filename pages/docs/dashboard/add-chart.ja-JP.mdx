# グラフを手動で追加する

### 1. 新しいグラフを作成する

!['Dashboard'](/image/connection-manage/creatDashboard.png)

左側の`Dashboard`をクリックし、「+」記号をクリックして新しいレポートを作成し、レポート名 (必須) とレポートの説明 (オプション) を入力します。


### 2. レポート情報を入力します

!['Dashboard2'](/image/connection-manage/dashboard2.png)

`Create Chart`をクリックし、ページ上の`Data allcation`をクリックして、生成されたレポートに対応するデータベースを選択します。

!['Dashboard3'](/image/connection-manage/dashboard3.png)

選択したら、対応する SQL ステートメントを記述し、`Run`をクリックして、`Back to Chart`をクリックします。

!['Dashboard4'](/image/connection-manage/dashboard4.png)

例: Employees テーブルの情報をクエリしてレポートを作成したいと考えています。


### 3.レポートの種類を選択します

!['Dashboard5'](/image/connection-manage/dashboard5.png)

ここでレポート タイプを選択し、選択したタイプに応じて対応する変数 (水平軸と垂直軸の座標、パーセンテージなど) を決定します。


### 4. レポートを保存します。

!['Dashboard6'](/image/connection-manage/dashboard6.png)

選択したら`save`をクリックして保存してください。

!['resultDashboard'](/image/connection-manage/resultDashboard.png)
