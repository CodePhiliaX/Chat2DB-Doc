# エディターの設定

ここでは、個人の好みや習慣に合わせてコード編集環境をカスタマイズできます。ここでは、エディターの設定にあるオプションのいくつかとその詳細な説明を示します。

!['エディタ'](/image/settings/editor.png)

## エディターのテーマ
Chat2DB は、インターフェースの色に対するさまざまなユーザーのニーズを満たすために、さまざまなエディター テーマを提供します。利用可能なテーマのリストは次のとおりです。

1. **Chat2DB-Dark**: これはデフォルトのダークテーマで、暗い環境での使用に適しており、目の疲労を効果的に軽減できます。
2. **Chat2DB-Light**: これは明るいテーマで、明るい環境での使用に適しており、コードが読みやすくなります。
3. **ドラキュラ**: これは、コード要素を強調表示するのに役立つ明るい対照的な色を備えたユニークなダークテーマです。
4. **Github-Dark**: 親しみやすく快適な Github Web サイトのダーク スタイルを模倣します。
5. **Github-Light**: 一貫性を維持するため、Github Web サイトと一致する軽量テーマ。
6. **Monokai-Bright**: モノカイの配色をベースにした、明るい色を使用した明るいテーマです。
7. **Monokai**: 古典的な Monokai の色の組み合わせは開発者に愛されています。
8. **One-Dark**: ソフトな視覚効果を備えたシンプルで一貫したダークテーマ。
9. **Solarized-Dark**: 視覚的な疲労を軽減することを目的とした、コントラストとカラーバランスに重点を置いた暗いテーマです。
10. **Solarized-Light**: Solarized-Dark に対応しますが、明るい背景を使用するため、明るい環境により適しています。
11. **Xcode**: Apple の Xcode 開発環境のネイティブ テーマで、iOS および macOS の開発者にはよく知られています。
12. **vs-dark**: Visual Studio のダーク テーマ。長時間のコーディングに適しています。
13. **hc-light**: 視力の悪いユーザーに便利な、ハイコントラストの明るい色のテーマ。
14. **hc-black**: ハイコントラストのダークテーマ。視力の悪いユーザーにも適しています。

新しいテーマを選択するには、ドロップダウン メニューをクリックし、リストから希望のテーマを選択します。選択すると、エディターがすぐに更新され、新しいテーマの外観が反映されます。いつでもこの設定ページに戻って、さまざまな作業環境や個人の好みに合わせてテーマを変更できます。

テーマが異なるとコードの可視性と読みやすさに影響を与える可能性があることに注意してください。そのため、いくつかのテーマを試して、最適なテーマを見つけることをお勧めします。

## フォントファミリー
フォント ファミリは、コードの表示に使用されるフォントを指定します。次のフォントの使用をお勧めします。

- **Monaco**: MacOS のデフォルトの等幅フォント。簡潔で読みやすい。
- **Consolas**: プログラミングやテキスト編集に適した、もう 1 つの人気のある等幅フォントです。
- **Courier New**: Windows のデフォルトの等幅フォント。プログラミングやテキスト編集に適しています。
- **JetBrains Mono**: 開発者向けに特別に設計された等幅フォントで、優れた読みやすさと美しさを提供します。


## フォント サイズと行の高さ
最も快適な読書体験を得るために、ニーズに応じてフォント サイズと行間隔を調整できます。フォント サイズの範囲は 12px ～ 24px、行間隔の範囲は行の高さの 1.0 ～ 3.0 倍です。

## 行番号を表示
このオプションをオンにすると、エディターの左側に行番号が表示され、コードの場所の追跡やエラーの参照が容易になります。

## ミニマップを表示
このオプションを有効にすると、エディターの右側にコードのサムネイルが表示され、ファイル構造をすばやく参照できるようになります。

## 自動行折り返し
長い行を水平方向にスクロールするのではなく、自動的に折り返す場合は、このオプションを有効にします。
