# エディターのフォント設定

!['font settings'](/image/settings/FontSettings.png)

このオプションを使用すると、個人の好みや使いやすさに合わせて SQL エディターのフォント サイズ パラメーターを調整できます。
デフォルトでは、エディターのフォント サイズは 13 ピクセルですが、これはお好みに合わせて変更できます。
設定の変更が完了したら、`Editor Font Size`をクリックしてコンソールのフォント サイズを編集します。
次回使用するときに新しい構成が有効になるように、忘れずに`Save`ボタンをクリックして設定を保存してください。
