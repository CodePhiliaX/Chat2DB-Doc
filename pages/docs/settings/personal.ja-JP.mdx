# 個人設定

!['first'](/image/settings/first.png)
左下隅のアバターをクリックし、`Profile Details`を選択して、個人情報ホームページにアクセスします。

## 個人情報

!['photo'](/image/settings/photo.png)

**アバターの変更**: ここで Chat2DB アカウントのアバターを変更できます。写真は JPG、PNG、GIF 形式のみをサポートし、1M を超えることはできません。

!['name'](/image/settings/name.png)

**名前の変更**: ここで Chat2DB 名を変更できます。システムが`Update success`というプロンプトを表示したら、変更が成功したことを意味します。


## メールボックスのリセット

!['email'](/image/settings/email.png)

**電子メールをリセット**: ここに電子メール情報を入力すると、後で電子メールを使用してすぐにログインできます。 認証コードを入力後、`Change email`ボタンをクリックすると、「4」にメールアドレスが表示されます。

## パスワードを変更する

!['password'](/image/settings/password.png)

**パスワードの変更**: ここにパスワードを入力すると、**電子メール + パスワード** を使用してログインするクイック ログイン方法として使用できます。

!['another'](/image/settings/another.png)

**注意事項**: パスワードを忘れた場合は、**電子メール + 確認コード** を使用してログインし、ここで新しいパスワードを変更してください。
