# 个人设置

!['first'](/image/settings/first.png)
点击左上角头像选择`Profile`进入个人信息主页:

## 个人信息

!['photo'](/image/settings/photo.png)

**更改头像**：您可在此处更改您的Chat2DB账号头像，照片仅支持JPG、PNG、GIF格式且最大不超过1M

!['name'](/image/settings/name.png)

**更改名称**：您可在此处更改您的Chat2DB名称，当系统提示`Update success`,表明修改成功！


## 重置邮箱

!['email'](/image/settings/email.png)

**重置邮箱**：在这里填写您的邮箱信息，后续可以通过邮箱快速登录。输入验证码后点击`Change email`按钮，会在“4”处出现显示您的邮箱

## 更改密码

!['password'](/image/settings/password.png)

**更改密码**：在此处填写好密码后可作为快捷登录方式，使用**邮箱+密码**的方式登录。

!['another'](/image/settings/another.png)

**温馨提示**：如果您忘记了密码，请先通过**邮箱+验证码**的方式登录，并在此处修改您的新密码
