# 编辑器设置

在这里，您可以自定义您的代码编辑环境，使其符合个人喜好和习惯。以下是您可以在编辑器设置中找到的一些选项及其详细解释。

!['editor'](/image/settings/editor.png)

## 编辑器主题（Editor Theme）
Chat2DB 提供多种编辑器主题，满足不同用户对界面颜色的需求。以下是可用的主题列表：

1. **Chat2DB-Dark**：这是默认的暗黑主题，适合在光线较弱的环境下使用，能有效减少眼睛疲劳。
2. **Chat2DB-Light**：这是一个亮色主题，适合在明亮环境中使用，使代码更易于阅读。
3. **Dracula**：这是一种独特的暗色主题，具有鲜明的对比色，有助于突出代码元素。
4. **Github-Dark**：模仿 Github 网站的暗色风格，熟悉而舒适。
5. **Github-Light**：与 Github 网站一致的亮色主题，保持一致性。
6. **Monokai-Bright**：基于 Monokai 配色方案的亮色主题，色彩鲜艳。
7. **Monokai**：经典的 Monokai 颜色搭配，深受开发者喜爱。
8. **One-Dark**：简洁的一致性暗色主题，视觉效果柔和。
9. **Solarized-Dark**：一种注重对比度和色彩平衡的暗色主题，旨在降低视觉疲劳。
10. **Solarized-Light**：与 Solarized-Dark 相对应，但采用亮色背景，更适合明亮环境。
11. **Xcode**：苹果 Xcode 开发环境的原生主题，对于 iOS 和 macOS 开发者来说很熟悉。
12. **vs-dark**：Visual Studio 的暗色主题，适合长时间编码。
13. **hc-light**：高对比度的亮色主题，便于视力不佳的用户使用。
14. **hc-black**：高对比度的暗色主题，同样适合视力不佳的用户。

要选择一个新的主题，请单击下拉菜单，然后从列表中选择您喜欢的主题。选择后，编辑器将立即更新以反映新主题的外观。您可以随时返回此设置页面来更改主题，以适应不同的工作环境和个人喜好。

请注意，不同的主题可能会影响代码的可见性和可读性，因此建议您尝试几个主题，找到最适合自己的那个。

## 字体家族（Font Family）
字体家族指定了用于显示代码的字体。我们推荐使用以下字体：

- **Monaco**：MacOS上的默认等宽字体，简洁易读。
- **Consolas**：另一种流行的等宽字体，适用于编程和文本编辑。
- **Courier New**：Windows上的默认等宽字体，适用于编程和文本编辑。
- **JetBrains Mono**：一种专为开发者设计的等宽字体，提供良好的可读性和美观度。


## 字体大小（Font Size）和行间距（Line Height）
您可以根据自己的需求调整字体大小和行间距，以获得最舒适的阅读体验。字体大小范围为12px至24px，行间距范围为1.0至3.0倍行高。

## 显示行号（Show Line Numbers）
开启此选项后，编辑器会在左侧显示行号，方便您跟踪代码位置和引用错误。

## 显示缩略图（Show Minimap）
启用此选项后，编辑器将在右侧显示代码的缩略图，让您快速浏览文件结构。

## 自动换行（Word Wrap）
如果您希望长行自动折行显示，而不是水平滚动，请启用此选项。
