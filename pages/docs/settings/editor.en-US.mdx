# Editor font settings

!['font settings'](/image/settings/FontSettings.png)

This option allows you to adjust the SQL Editor font size parameters to suit your personal preferences and comfort.
By default, the editor font size is 13px, but you can change this value to suit your preferences.
Click `Editor Font Size` to edit the console font size. Once you have finished changing the settings,
remember to click the `Save` button to save your settings so that the new configuration will take effect the next time you use it.
