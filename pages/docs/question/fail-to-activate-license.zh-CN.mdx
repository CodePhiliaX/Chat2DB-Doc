# Local 授权码激活失败

在激活 Chat2DB Local 时，如果遇到授权码激活失败的情况，通常有以下几种可能的原因。请根据具体情况检查并采取相应的解决措施。

## 1. 激活码不正确

输入的激活码错误，可能包含拼写错误或格式不正确。

**解决方法：**

- 请仔细核对您输入的激活码是否与购买时收到的激活码完全一致。
- 确认没有多余的空格、字母大小写是否正确。
- 如果仍然无法激活，请联系客服支持以获取进一步的帮助。

## 2. 绑定设备数量已达上限

每个 License 默认可绑定的设备数量为2台。请确认您的 License 是否已达到绑定设备的数量上限。

## 3. 该设备之前未正常解绑 License

之前您的设备可能未正确解绑 License，请确认您是否通过正常流程进行了解绑。参考：[Local 授权码激活/解绑](/docs/start-guide/activate-unbind-license)

**解决方法：**

**第一步：使用旧 License 重新激活并解绑。**

  1. 前往**左上角设置 -> 激活码**页面。
  ![激活码弹窗](/image/start-guide/license3.jpg)
  2. 点击 **“激活”** 输入旧的 License 进行激活。
  ![激活码弹窗](/image/start-guide/license2.jpg)
  3. 点击 **“解绑”** 输入旧的 License 进行解绑。
  ![激活码弹窗](/image/start-guide/unbind.jpg)
  ![激活码弹窗](/image/start-guide/unbind1.jpg)

**第二步：使用新的 License 进行激活。**

  1. 解绑完成后，再次进入**左上角设置 -> 激活码**页面。
  ![激活码弹窗](/image/start-guide/license3.jpg)
  2. 点击 **“激活”** 按钮，输入新的 License 并点击“激活”。
  ![激活码弹窗](/image/start-guide/license2.jpg)
