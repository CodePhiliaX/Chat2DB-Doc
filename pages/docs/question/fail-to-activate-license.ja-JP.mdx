# Localライセンスのアクティベーション失敗

Chat2DB Localのアクティベーション中にライセンスのアクティベーションが失敗した場合、通常以下のいくつかの原因があります。具体的な状況に応じて確認し、適切な対処を行ってください。

## 1. アクティベーションコードが不正確

入力されたアクティベーションコードが間違っている可能性があります。スペリングミスやフォーマットが正しくない場合があります。

**解決方法：**

- 購入時に受け取ったアクティベーションコードと入力したコードが完全に一致しているか、慎重に確認してください。
- 餘計なスペースがないこと、および大文字小文字が正しいことを確認してください。
- それでもアクティベーションできない場合は、カスタマーサポートに連絡して追加のサポートを受けてください。

## 2. バインドされたデバイスの最大数に達しました

各ライセンスはデフォルトで最大2つのデバイスにバインドすることができます。ライセンスがバインドされたデバイスの最大数に達していないか確認してください。

## 3. 以前のライセンスが正常に解除されていない

以前のデバイスが正常にライセンスから解除されていない可能性があります。正常な手順で解除したかどうか確認してください。参照: [Local ライセンスのアクティベーション/解除](/docs/start-guide/activate-unbind-license)

**解決方法：**

**ステップ1: 古いライセンスを使用して再アクティベーションし、解除する。**

  1. **左上の設定 -> アクティベーションコード**ページへ進んでください。
  ![Activation Code Pop-up](/image/start-guide/license3-en.jpg)
  2. **「アクティベーション」**をクリックし、古いライセンスを入力して再アクティベーションを行ってください。
  ![Activation Code Pop-up](/image/start-guide/license2-en.png)
  3. **「解除」**をクリックし、古いライセンスを入力して解除を行ってください。
  ![Unbind](/image/start-guide/unbind-en.jpg)
  ![Unbind](/image/start-guide/unbind1-en.jpg)

**ステップ2: 新しいライセンスを使用してアクティベーションする。**

  1. 解除後、再度**左上の設定 -> アクティベーションコード**ページへ進んでください。
  ![Activation Code Pop-up](/image/start-guide/license3-en.jpg)
  2. **「アクティベーション」**ボタンをクリックし、新しいライセンスを入力して「アクティベーション」をクリックしてください。
  ![Activation Code Pop-up](/image/start-guide/license2-en.png)
