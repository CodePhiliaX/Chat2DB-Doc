## 常见问题
### Q: 为什么我科学上网了openai还是用不了
A: 如果网页gpt可以正常聊天，说明你的VPN只对浏览器生效了，不是全局的，并没有对部分Java应用生效，在chat2db客户端中手动配置代理即可。

     > 需要配置：
     > * HTTP Proxy Host 一般为127.0.0.1
     > * HTTP Proxy Port VPN软件代理的本地端口号 小猫咪为7890
     > * Api Host 如果你直接用的openai的服务没有做中间代理那么不需要填写



### Q: 前端页面白屏
A: 因为一些前端本地存储的值，导致新老版本不兼容，可以尝试打开控制台清除localstorage，`后续版本再v2.0.2出现不兼容的情况，我们会自动清除前端本地存储内容`。

### Q: 前端运行yarn安装依赖，卡在Electron无响应
A: 1. 可以尝试修改镜像      2.修改.npmrc中Electron镜像地址（[参考文章](https://www.cnblogs.com/makalochen/p/16154510.html#%E8%A7%A3%E5%86%B3%E5%8A%9E%E6%B3%95)）

### Q: 新建链接之后，表是空的
A: 需要在链接顶部切换到对应的数据库即可，参考这里 -> https://github.com/codePhiliaX/Chat2DB/issues/439

### Q: 可视化修改数据为什么报错：Each SOL statement should update no more than one record. Please use a unlque key for updates。
A: 如果你的表没有唯一键，并且有N条长的一模一样的数据，这时你修改其中一条数据，在技术层面没办法知道你期望修改的是哪条。
### Q: AI功能报错：出现异常,请在帮助中查看详细日志：CHAT2DB_SERVICE_ERROR
A: 表字段过多，可以少选几张表试试
