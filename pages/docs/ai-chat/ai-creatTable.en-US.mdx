# AI table creation

AI table creation automatically matches the most appropriate field type and comment index based on the table name and column name entered by the user, and automatically generates SQL table creation statements.

!['table-create'](/image/AI-image/table-copilot.gif)

## Specific operations

### 1. First open the table creation interface

!['table-create'](/image/AI-image/create-table01.png)

### 2. Click the `Table Copilot` button

!['table-create'](/image/AI-image/create-table02.png)

### 3. Enter the custom table name and column name and press Enter to execute

!['table-create'](/image/AI-image/create-table03.png)

### 4. View the generated SQL table creation statement and click `Run` to execute the SQL statement

!['table-create'](/image/AI-image/create-table04.png)

### 5. Execution successful!

!['table-create'](/image/AI-image/create-table05.png)

### 6. In the existing table structure, we can also add fields according to the needs, click the `Space` space bar to continue

!['table-create'](/image/AI-image/create-table06.png)

### 7. Success!

!['table-create'](/image/AI-image/create-table07.png)
