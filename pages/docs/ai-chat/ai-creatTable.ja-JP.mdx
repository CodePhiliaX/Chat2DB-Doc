# AI がテーブルを作成

AI テーブル作成では、ユーザーが入力したテーブル名と列名から、最も適切なフィールド タイプとアノテーション インデックスを自動的に照合し、SQL テーブル作成ステートメントを自動的に生成します。

!['テーブル作成'](/image/AI-image/table-copilot.gif)

## 特定の操作

### 1. まずテーブル作成インターフェイスを開きます。

!['テーブル作成'](/image/AI-image/create-table01.png)

### 2. `Table Copilot`ボタンをクリックします。

!['テーブル作成'](/image/AI-image/create-table02.png)

### 3. カスタマイズしたテーブル名と列名を入力し、Enter キーを押して実行します。

!['テーブル作成'](/image/AI-image/create-table03.png)

### 4. 生成された SQL テーブル作成ステートメントを表示し、`Run`をクリックして SQL ステートメントを実行します。

!['テーブル作成'](/image/AI-image/create-table04.png)

### 5. 実行は成功しました!

!['テーブル作成'](/image/AI-image/create-table05.png)

### 6. 既存のテーブル構造に、必要に応じてフィールドを追加することもできます。`Space`をクリックして続行します。

!['テーブル作成'](/image/AI-image/create-table06.png)

### 7. 成功しました!

!['テーブル作成'](/image/AI-image/create-table07.png)
