# AI SQL スマート プロンプト

- **リアルタイムのコード提案と補完**: ユーザーが SQL クエリの作成を開始すると、SQL Copilot はリアルタイムの構文提案とコード スニペットを提供して、ユーザーがクエリの作成をより迅速に完了できるようにします。
このツールは、テーブル名、フィールド名、関数などのユーザーのクエリ ステートメントのコンテキストに基づいて関連する提案を提供し、スペル ミスを減らし、クエリの精度を向上させることができます。

- **SQL コードを自動的に生成**: ユーザーは自然言語でクエリまたは質問したいデータを直接記述することができ、SQL Copilot は対応する SQL コードを自動的に生成します。
ユーザーは、サンプル データと期待される結果を提供することで、ツールに適切な SQL ステートメントを生成させることもできます。

- **統合と互換性**: SQL Copilot は、MySQL、PostgreSQL、Oracle、SQL Server などの複数のデータベース システムをサポートし、さまざまな環境で一貫した支援を保証します。
時間の経過とともに、SQL Copilot はユーザーの習慣や好みを学習し、よりパーソナライズされた推奨事項やサポートを提供します。
