# Excel 智能问答

Excel 智能问答功能不仅能够帮助用户通过自然语言查询快速获取所需的数据信息，还支持根据用户的请求自动生成各种类型的智能图表，如折线图、直方图、饼图等。
这一特性使得用户能够直观地理解数据趋势和分布情况，而无需手动设置图表类型或调整数据系列。
借助这项技术，即使是不具备专业技能的用户也能轻松创建专业的数据分析图表，从而更好地分析数据、展示成果并辅助决策制定过程。

!['excel'](/image/AI-image/excel001.gif)

## 1. 创建新聊天

点击左侧`AI对话`，进入对话界面。点击加号"+"按钮以开始一个新的对话。

!['excel'](/image/AI-image/excel01.png)

新建对话选择`表格类型`, 选择上传excel文档。

!['excel'](/image/AI-image/excel02.png)

双击对话名称可以修改对话名称。现在我们就可以开始针对于`Excel`进行智能问答了。

!['excel'](/image/AI-image/excel03.png)
