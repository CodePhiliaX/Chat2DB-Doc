# AI 建表

AI辅助的智能表结构设计功能允许用户通过简单的输入，如表名和列名，来快速生成完整的SQL建表语句。
这一智能化过程会自动识别并推荐最适合的字段数据类型、添加适当的注释以及创建必要的索引，
从而极大地简化了数据库表的设计与创建流程，提高了开发效率并确保了表结构的合理性与优化性。

!['table-create'](/image/AI-image/table-copilot.gif)

## 具体操作

### 1.首先打开创建表界面

!['table-create'](/image/AI-image/create-table01.png)

### 2. 点击`Table Copilot`按钮

!['table-create'](/image/AI-image/create-table02.png)

### 3. 输入自定义的表名及列名并回车执行

!['table-create'](/image/AI-image/create-table03.png)

### 4. 查看生成的SQL建表语句，并点击`Run`执行sql语句

!['table-create'](/image/AI-image/create-table04.png)

### 5. 执行成功！

!['table-create'](/image/AI-image/create-table05.png)

### 6. 在已存在的表结构中，我们还可以根据需求添加字段，点击`Space`空格键继续

!['table-create'](/image/AI-image/create-table06.png)

### 7. 成功！

!['table-create'](/image/AI-image/create-table07.png)
