# 安装指南

我们强烈建议您关闭所有打开的应用程序。这将有助于确保顺利安装。

Chat2DB 支持以下操作系统：

- Windows: windows x64.
- MacOS ：macOS 英特尔/苹果M 芯片。
- Linux：打包为.AppImage，支持x86_64和arm64。

## Windows

1. 下载 Chat2DB Windows 版本的`.exe`安装程序。

2. 双击运行`.exe`文件启动安装向导。

3. 按照安装向导的提示完成安装过程。


## MacOS

1. 下载 Chat2DB macOS 版本。

2. 打开`.dmg`文件。

3. 将 Chat2DB 拖到您的应用程序文件夹中进行安装。


## Linux

1. 点击相应的下载链接（`.ApplImage x86` 或 `.ApplImage arm64`），下载应用程序文件。

2. 在终端中导航至包含已下载应用程序文件的目录。

3. 给应用程序文件添加执行权限：

```bash
chmod +x <downloaded_file>
```
其中 `<downloaded_file>` 是您下载的应用程序文件名（例如 `chat2db-linux-x86.applimage`）。

4. 运行应用程序：

```bash
./<downloaded_file>
```

应用程序应该现在就可以运行了。请确保您的 Linux 发行版允许从非标准存储库安装应用程序，或者您可能需要调整安全设置以允许运行此类应用程序。

## 自动更新

此功能仅在 `Windows` 和 `macOS` 上可用。

Chat2DB 会定期自动检查新版本。如果未禁用该检查并且有新版本可用，则会提示您决定是否升级。如果您选择升级，安装程序将在完成后自动下载并启动。

默认情况下，Chat2DB 配置为自动检查更新并在新版本可用时通知您。
更新通常是基于补丁的：它们适用于现有安装，并且只需要您重新启动应用。
但是，有时补丁更新不可用，必须在[官网](https://chat2db-ai.com/download)安装新版本的 Chat2DB。

如果 Chat2DB 在本地网络之外没有 HTTP 访问权限，它将无法检查更新并应用补丁。
在这种情况下，您必须下载新版本的 Chat2DB 并手动安装它们。

> 注意： Chat2DB 本身会正常关闭以避免数据丢失。
