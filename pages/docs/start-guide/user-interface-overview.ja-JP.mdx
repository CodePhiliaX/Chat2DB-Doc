# ユーザー インターフェイスの概要

Chat2DB ウィンドウには、メニュー バー、ツールバー、ナビゲーション バー、1 つ以上のエディターとビューを含むワークスペース、およびステータス バーが含まれています。

!["user-interface"](/image/start-guide/user-interface.png)

## メニューバー

デフォルトでは、メニュー バーには次のメニューが含まれています。

- 「Chat2DB Pro」メニューには、**再起動**および**終了**のメニュー項目が含まれています。
- 「編集」メニューには、**切り取り、コピー、貼り付け、すべて選択、元に戻す、やり直し** 操作など、アクティブな要素のグローバル コマンドが含まれています。
- 「表示」メニューには、**更新、ズームインおよびズームアウト、ズームビュー、および全画面** 操作のためのメニュー項目が含まれています。
- 「ヘルプ」メニューには、**公式 Web サイト、ドキュメント**へのリンク、**ログとコンソール**を確認するためのメニュー項目が含まれています。

## ツールバー

ツールバーには、**データベース接続の作成**、**更新**、**検索**の最も基本的なボタンが含まれています。

詳細については、[Toolbar Usage](https://docs.chat2db.ai/ja-JP/docs/connection-manage/database-operations#%E7%AE%A1%E7%90%86%E3%83%87%E3%83%BC%E3%82%BF%E3%83%99%E3%83%BC%E3%82%B9%E3%82%92%E8%A1%A8%E7%A4%BA-%E5%B7%A6%E3%81%8B%E3%82%89%E5%8F%B3%E3%81%B8  )をご覧ください。

## ナビゲーション バー

ナビゲーション バーには次の項目が含まれています。

1. **個人情報**: 個人情報またはアカウント設定を表します。クリックすると、アバター、ユーザー名などの個人情報を表示および編集したり、パスワードやその他の関連設定を変更したりできます。個人情報の詳細については、[Personal Information](https://docs.chat2db.ai/ja-JP/docs/settings/personal)をご覧ください。

2. **ワークスペース**: ワークスペースは、ファイルやデータベースの内容を操作できるウィンドウです。

3. **ダッシュボード**: ダッシュボードには、主要業績評価指標やその他の重要な情報を示す概要ビューが表示されます。ニーズや好みに応じて、さまざまなタイプのダッシュボードから選択して、さまざまな形状のグラフを取得できます。ダッシュボードの詳細については、「[Dashboard](https://docs.chat2db.ai/ja-JP/docs/dashboard/Common-operations-dashboard)」を参照してください。

4. **AI チャット**: AI チャットは、人工知能アシスタントと対話する機能を指す場合があります。ここでは、SQL について質問したり、ヘルプを得たり、関連情報を入手したりできます。

5. **プラグイン**: プラグインを使用すると、アプリケーションの機能を拡張できます。さまざまなプラグインをインストールして有効にすることで、特定のニーズに合わせてアプリケーションを調整できます。特定のプラグインの詳細については、「[Plugin](https://docs.chat2db.ai/ja-JP/docs/plugin/DBMotion)」を参照してください。

6. **グローバル更新**: グローバル更新ボタンは、ページ全体のコンテンツを更新するために使用されます。すべてのデータを再読み込みしたい場合にこの機能を使用できます。

7. **通知**: 通知アイコンには、未読メッセージの数が表示されます。アイコンをクリックすると、新機能やリマインダーなど、アプリからのさまざまな通知が表示されます。

8. **フィードバック**: フィードバック機能を使用すると、開発者にコメントを送信したり、問題を報告したりできます。これは、製品とサービスの品質の向上に役立ちます。フィードバックの詳細については、「[Submit Feedback](https://docs.chat2db.ai/ja-JP/docs/feedback/give-feedback)」を参照してください。



## ショートカット バー

ショートカット バーには、テーブル情報、実行 SQL ログ、保存されたコンソール、およびエクスポートの進行状況へのショートカットが表示されます。

!["情報"](/image/start-guide/ddlnfo.png)
!["savedSql"](/image/start-guide/savedSql.png)

コンソールのショートカット キーを保存します。
> 勝利: `Ctrl + S`

> マック: `Command + S`

コンソールの保存の詳細については、[Save Console Information](https://docs.chat2db.ai/ja-JP/docs/connection-manage/Using-toolbar#%E3%82%B3%E3%83%B3%E3%82%BD%E3%83%BC%E3%83%AB%E3%82%92%E4%BF%9D%E5%AD%98) を参照してください。

!["savedConsole"](/image/start-guide/savedConsole.png)

## ワークスペース

エディターは、ファイルやデータベースの内容を操作できるウィンドウです。ワークスペースでは複数のビューとエディターを同時に開くことができますが、一度にアクティブにできるのは 1 つだけです。

テーブル ビューとエディタは両方とも、別個のウィンドウまたはタブとして表示でき、タブ付きウィンドウで他のビュー/エディタと重ねて表示できます。
次の図は、タブ付きウィンドウのタイトル バーを示しています。非表示のタブのリストを表示するには、⬇️ アイコンをクリックすると、リストが表示され、検索可能になります。

!["追加"](/image/start-guide/addmore.png)


### クエリコンソール

新しいコンソールのショートカット キーを作成します。
> 勝利: `Ctrl + Shift + L`

> Mac: `Command + Shift + L`

クエリ コンソールを使用して、SQL コードの読み取り、書き込み、実行を行います。

!["コンソール"](/image/start-guide/console01.png)

ビューまたはエディタを閉じるには、「閉じる」ボタンをクリックするか、ビュー/エディタのタイトル バーを右クリックしてコンテキスト メニューのいずれかのオプションをクリックします (ウィンドウの構成によって異なります)。

- 閉じる: アクティブなウィンドウまたはタブ付きウィンドウ内のタブを閉じます。
- その他を閉じる: (タブ付きウィンドウでタブとして表示されるエディタおよびビューの場合) - 現在のタブ付きウィンドウ内のアクティブなタブを除くすべてのタブを閉じます。
- すべて閉じる: タブ付きウィンドウのすべてのタブを閉じます (ウィンドウを閉じます)。

### データエディターおよびビューアー

データ エディターを使用して、データベース オブジェクト、CSV ファイル、およびクエリ結果のデータを並べ替え、フィルター処理、追加、編集、および削除します。

!["コンソール"](/image/start-guide/console02.png)

テーブル ビュー ウィンドウのサイズを変更できます。サイズを変更するには、ウィンドウの境界線が両方向矢印に変わるまでカーソルをウィンドウの境界線の上に置き、端をクリックして目的のサイズまでドラッグします。

### 「クエリ結果」タブ

SQL を実行するショートカット キー:
> 勝利: `Ctrl + R`

> マック: `Command + R`

サービス ツール ウィンドウの [結果] タブでクエリの結果セットを表示します。

!["コンソール"](/image/start-guide/console03.png)

## ステータスバー

メイン ウィンドウの下部にあるステータス バーの右側には、最近のエクスポートの進行状況が表示されます。ショートカット バーの `進行状況のエクスポート` をクリックし、`すべて表示` をクリックして開きます。
