# オープンソースと商用化

私たちはオープンソースの力とそれがもたらす無限の可能性を信じています。オープンソースを通じて、世界中のコミュニティに利益をもたらす貴重なツールを共同作成できます。 Chat2DB の作者は EasyExcel の作者でもあり、私たちが 7 年間維持してきた非常に人気のあるオープン ソース プロジェクトです。これは、私たちがオープン ソースの精神に注力し、重視していることを完全に証明しています。ユーザーのニーズ。
Chat2DB がオープンソースになってから 1 年間、私たちは GitHub のホット プロジェクト リストに何度も掲載され、多くのユーザーから貴重なフィードバックとサポートを受け取りました。積極的にご参加いただきましたすべての寄稿者およびサポーターの皆様に心より感謝申し上げます。私たちは、Chat2DB が幅広いユーザー コミュニティにより良いサービスを提供できる優れたデータ管理ツールとなるよう、Chat2DB の機能とパフォーマンスの向上に今後も懸命に取り組んでいきます。

同時に、製品の開発と反復を継続したい場合は、より多くのリソースのサポートと、それを継続的に保守できるチームが必要です。商用モデルなしではこれを行うのは困難です。私たちは、次のような多くの有名なオープン ソース プロジェクトを見てきました。 Cglib は多くの人に使用されているかもしれませんが、現在はメンテナンスされていないことをご存知ですか?私たちは Chat2DB がそのようなプロジェクトになることを望んでいません。私たちは、Chat2DB が今後も開発を続け、反復を続け、皆様にサービスを提供し続けることを願っています。そこで、より多くの機能、より良いサービス、より良いサポートを提供するために、製品版を開始することにしました。オープンソース バージョンと商用バージョンは相互に補完し合い、オープンソース ユーザーが増えると商用ユーザーも増えるため、開発を続けることができます。商用ユーザーがいると、オープンソースをより適切にサポートできます。

あなたは心配しているかもしれません: オープンソースバージョンをあきらめるか?一部のユーザーがオープンソース バージョンのサポートについて懸念を抱いていることは理解しています。ここで、私たちは Chat2DB のオープンソース バージョンを揺るぎなく保守、更新、最適化することを厳粛に約束します。私たちはオープンソース コミュニティの力を強く信じており、オープンソース バージョンの長期的な開発と安定した運用を確保するためにリソースを投資し続けます。

弊社のオープンソース コミュニティに参加し、知恵やスキルを提供することに興味がある方は、大歓迎です。 GitHub リポジトリ ( https://github.com/chat2db/chat2db ) にアクセスしてプル リクエスト (PR) を送信するか、Chat2DB@ch2db.com に電子メールを送信して問い合わせることができます。すべての投稿者に対して、感謝の気持ちを表すために適切な報酬を提供します。
