# オープンソースと商用化
私たちはオープンソースはとても素晴らしいことだと考えており、より多くの人に利益をもたらすツールを作成できることは非常に有意義です。 Chat2DB の作者は Easyexcel の作者でもあります。Easyexcel は非常に人気のあるオープン ソース プロジェクトであり、7 年間維持されており、Github には 30,000 個以上のスターが付いています。 また、Chat2DB も人気のオープンソース プロジェクトになることを期待しています。 Chat2DB がオープンソースになってから 1 年間、私たちは Github の人気プロジェクトリストに何度も名を連ねており、ユーザーからの多くのフィードバックや、ユーザーからの貢献やサポートもいただきました。継続的な努力により、Chat2DB がすべての人に役立つより良いツールになるようにしていきます。

同時に、製品の開発と反復を継続したい場合は、より多くのリソースのサポートと、それを継続的に保守できるチームが必要ですが、商用モデルなしではこれを行うのは困難です。 私たちは、次のような多くの有名なオープン ソース プロジェクトを見てきました。 Cglib は多くの人に使用されているかもしれませんが、現在はメンテナンスされていないことをご存知ですか? 私たちは Chat2DB がそのようなプロジェクトになることを望んでいません。 私たちは、Chat2DB が今後も開発を続け、反復を続け、皆様にサービスを提供し続けることを願っています。 そこで、より多くの機能、より良いサービス、より良いサポートを提供するために、製品版を開始することにしました。 オープンソース版と商用版は相互補完しており、オープンソースのユーザーが増えれば商用ユーザーも増え、開発を続けることができます。 商用ユーザーがいると、オープンソースをより適切にサポートできます。

オープンソース版を諦めるのではないかと心配しているかもしれません。 私たちは、オープンソース バージョンの維持、オープンソース バージョンの更新、およびオープン ソース バージョンの最適化を継続することをお約束します。 オープンソースに参加したい友人は、Github https://github.com/chat2db/chat2db を通じて PR を送信するか、Chat2DB@ch2db.com までご連絡ください。貢献者には一定の報酬を与えます。
