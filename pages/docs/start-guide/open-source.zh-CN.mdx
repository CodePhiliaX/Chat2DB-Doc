 # 开源&商业化

 我们坚信开源的力量及其带来的无限可能性。通过开源，我们能够共同创造有价值的工具，惠及全球社区。Chat2DB的作者也是EasyExcel的作者，Easyexcel是一个非常受欢迎的开源项目，我们坚持维护了7年时间，Github 有超过 30k 的 star，这充分证明了我们对开源精神的承诺以及对用户需求的关注。
 自 Chat2DB 开源以来的一年时间里，我们已经多次荣登 GitHub 热门项目榜单，并收到了很多用户的宝贵反馈和支持。在此，我们衷心感谢每一位贡献者和支持者的积极参与。我们将继续致力于提升 Chat2DB 的功能和性能，确保其成为一款卓越的数据管理工具，以更好地服务于广大用户社群。

 于此同时，想要让一个产品不停的发展和迭代，我们需要更多的资源支持，需要一个能够持续维护的团队，没有一套商业化的模式，是很难做到的。我们看到很多大名鼎鼎的开源项目，如：Cglib 可能很多人都在用，但是你知道它已经无人维护了吗？我们不希望Chat2DB 也成为这样的项目。我们希望Chat2DB 能够持续的发展，持续的迭代，持续的服务于大家。所以我们决定推出商业化的版本，我们会提供更多的功能，更好的服务，更好的支持。开源版本和商业版本相辅相成，有更多的开源用户，也会有更多的商业用户，这样我们才能够持续的发展。有了商业用户我们可以更好的反哺开源。

 大家可能会担心：那你们会不会把开源的版本给放弃了？我们理解部分用户可能对于开源版本的支持有所顾虑。在这里，我们郑重承诺：我们将坚定不移地维护、更新及优化开源版本的 Chat2DB。我们坚信开源社区的力量，并将持续投入资源以确保开源版本的长期发展和稳定运行。

 如果您有意加入我们的开源社区，贡献您的智慧与技能，我们热烈欢迎您！您可以通过访问我们的 GitHub 仓库（ https://github.com/codePhiliaX/chat2db ） 提交 Pull Request (PR)，或者发送邮件至 Chat2DB@ch2db.com 与我们取得联系。对于所有贡献者，我们将提供适当的奖励以表达我们的感激之情。
