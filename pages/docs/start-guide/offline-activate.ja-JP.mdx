# 完全オフラインモード

## デバイスIDの取得
オフラインアクティベーションに進み、デバイスIDを自動的に取得します

Webページでデバイス証明書を取得してください

<div style={{maxWidth: '500px'}}>
![デバイスIDの取得](/image/start-guide/device-id.png)
</div>


## デバイス証明書の取得
上記で取得したデバイスIDをコピーし、
デバイス証明書を取得してください http://app.chat2db.ai/settings/deviceCer
![デバイス証明書の取得](/image/start-guide/device-cer.png)


以下の証明書をコピーし、クライアントでアクティベーションを行ってください

![デバイス証明書の取得](/image/start-guide/device-cer-dialog.png)
</div>
