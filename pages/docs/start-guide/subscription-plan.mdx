# 订阅计划

## 订阅流程

> 有关我们的会员计划的更多信息，请参阅[会员协议](https://docs.chat2db.ai/docs/service/member)

首先，您需要登录您的 Chat2DB 账号。

如果您订阅了套餐，则可以在订阅期内登录您的 Chat2DB 账号来使用 Chat2DB Pro。

您可以**点击头像**或者**查看个人信息**来查看您的订阅到期时间。会员到期后可再次续订计划。

!["plan"](/image/start-guide/plan01.png)

订阅计划分为以下两类：

- **个人版会员**：适合个人开发者、数据分析师或小型项目团队。输入折扣码还能享受优惠。

!["plan"](/image/start-guide/plan02.png)


- **团队版会员**：适合中大型团队或企业级用户。输入折扣码还能享受优惠。

!["plan"](/image/start-guide/plan03.png)

**若订阅成功后系统无响应，重启应用即可！**
