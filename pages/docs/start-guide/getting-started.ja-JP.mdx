# クイックスタート

## 📖 紹介する

Chat2DB は、AI ファーストのデータ管理、開発、分析ツールです。AIGC (人工知能生成コード) 機能を核としており、自然言語から SQL への変換、SQL から自然言語への変換、レポートの自動生成も可能で、担当者の効率を大幅に向上させます。 。 1つの製品でデータ管理、データ開発、データ分析の機能を実現でき、SQLを理解していなくてもビジネスデータのクエリやレポートの生成を迅速に行うことができます。

## 🚀 クイック起動

このクイック スタート ガイドでは、重要な概念を紹介し、Chat2DB の最初のステップを支援します。

当社の製品は、Web ページとクライアントの 2 つの使用方法を提供します。 Web ページはすぐに使用でき、クライアントはよりスムーズなユーザー エクスペリエンスを提供します。同時に、一部のプライベート データベースやローカル データにパブリック ネットワークからアクセスできない場合、クライアントはデータベースにローカルに接続して、プロキシを必要とせずに直接送信できるため、シンプルで便利であり、シナリオに応じて Web ページの使用とクライアントの使用を柔軟に切り替えることができます。
以下に製品の使用方法を説明します。

WEBアクセスアドレス: [https://app.chat2db.ai](https://app.chat2db.ai), クライアントダウンロードアドレス: [https://chat2db.ai/download](https://chat2db.ai/download)

Web バージョンを使用する場合は、この手順を無視して、[アプリケーション アドレス](https://app.chat2db.ai) に直接アクセスできます。

クライアント ダウンロード アドレス: [クライアント ダウンロード アドレス](https://chat2db.ai/download) 対応するシステムを選択してダウンロードしてインストールするだけです [Windows](https://docs.chat2db.ai/ja-JP/docs/start-guide/installation-guide#windows)、[Mac](https://docs.chat2db.ai/ja-JP/docs/start-guide/installation-guide#macos) 、[Linux](https://docs.chat2db.ai/ja-JP/docs/start-guide/installation-guide#linux) をサポートしています。

## 第三步：データベース接続を作成する

データベース接続は Chat2DB を使用するための最初のステップです。後続の操作を実行するにはデータベースに接続する必要があります。 左側のデータベース接続をクリックし、左下隅の 新しい接続 をクリックして、使用するデータベースを選択し、データベース接続情報を入力して、保存 をクリックします。 現在サポートしておりますMysql、Postgresql、SQLServer、DB2、 Oracle、SQLServer、Oceanbase、Clickhouse、Redis、H2、SQLite、DM、KingBase および他のデータベース。
### 1. 新しい接続を作成します
!['newConnection-connection'](/image/connection/newConnection.png)

左側の データベース ボタンをクリックし、新しい接続 を選択し、必要なデータベースの種類を選択し、データベース接続情報を入力して、保存 をクリックします。

### 2. データベース接続情報を入力します。

!['newConnection-connection'](/image/connection/connectioninfo.png)

- `Name`：接続名をカスタマイズすると覚えやすくなります
- `Env`：接続環境、テスト、本番などを選択して、異なる環境での接続を区別します
- `Host`: Mysql サーバー アドレス。IP またはドメイン名にすることができます。
- `Port`: Mysql サーバー ポート、デフォルトは 3306
- `Authentication`: Mysql 認証方法、ユーザー名とパスワード、または認証は不要です
- `User`: MySQL ユーザー名
- `Password`: MySQL パスワード
- `Database`: Mysql データベース名、オプション。入力しない場合は、デフォルトのデータベースに接続します。
- `URL`: Mysql 接続 URL、オプション、入力しない場合は、上記の情報に基づいて自動的に生成されます
- `Driver`: Mysql ドライバー (オプション)。入力しない場合は、URL に基づいて自動的に識別されます。手動で選択することもできます。手動で選択すると、ドライバー設定項目が表示されます。
- `SSH`: SSH 接続を使用するかどうか、オプション。未入力の場合、デフォルトで無効になります。選択すると、SSH 設定項目が表示されます。
- `Advanced Configuration`: 詳細設定、オプション、未入力の場合はデフォルトでは使用されません。選択後に詳細設定項目が表示されます。

クライアント上のデータベースごとに異なるドライバーをダウンロードする必要があり、SSH 接続が必要な場合は、SSH 情報を入力する必要があります。


## 第四步：接続のテスト

['ssh'](/image/connection/test.png)

 左下のテストボタンをクリックして、接続が正常かどうかをテストします。接続が正常であれば「接続成功」と表示され、そうでなければ「接続失敗」と表示されます。エラーメッセージに基づいて接続情報を調整することができます。最後に保存をクリックします。

## 第五步：AIデータセットの作成

接続を作成した後、AIデータセットを設定するかどうかを尋ねるポップアップウィンドウが表示されます。AIデータセットを設定することで、AIがあなたのデータベーステーブル構造をより良く理解し、より正確な回答を生成することができます。これはChat2DBの主要な特長の一つです。

'確認' をクリックして設定を開始します。

[aidata](/image/start-guide/aidata1.png)

1. 右上の角で、AIデータセットを作成する必要があるデータベースを選択します。
2. 左側で同期したいデータベーステーブルにチェックを入れます。
3. 「AIデータセットへ同期」をクリックして作成を完了します。

[aidata](/image/start-guide/aidata2.png)

## 第六步：Chat2DBデータベース開発を使用する

接続が成功したら、接続をダブルクリックするか、右クリックして接続に入ることができます。その後、Chat2DBでデータベースの追加・削除・更新・検索操作を行うことができます。また、私たちのAIGC機能を使用して自然言語で直接データベースを問い合わせることも、SQLを自然言語に変換することもできます。

  ### データベース管理
  新しく作成した接続をダブルクリックして、データベース管理画面に入ります。ここでデータベースの追加・削除・更新・検索操作を行うことができます。
  ['databasemanage'](/image/connection-manage/databasemanage.png)

  ### テーブル管理

  手動でデータベーステーブルを作成することも、またはAIでワンクリックテーブル作成を行うこともできます。

  #### 1. AIによるワンクリックテーブル作成

  Table Copilotをクリックし、カスタムのテーブル名と列名を順に入力しエンターを押します。

  ['table-create'](/image/AI-image/create-table03.png)

  RunをクリックしてSQL文を実行し、作成を完了します。

  ['table-create'](/image/AI-image/create-table04.png)

  #### 2. 手動でのテーブル作成

  基本情報の入力：
  - `Table Name`: テーブル名
  - `Comment`: テーブルコメント
  - `Character set`: 文字コード
  - `Engine`: ストレージエンジン
  - `Increment value`: オートインクリメント開始値
  ['tables'](/image/connection-manage/create_t_base.png)

  列情報の入力：
  - `Column Name`: 列名
  - `Type`: 型
  - `Size`: 長さ
  - `Nullable`: NULL許容
  - `Key`: 主キー
  - `Comment`: コメント
  ['tables'](/image/connection-manage/create_t_column.png)

  インデックス情報の入力：
  - `Index Name`: インデックス名
  - `Column`: 列名
  - `Type`: インデックス型
  - `Include column`: 含まれる列
  - `Comment`: コメント
  ['tables'](/image/connection-manage/create_t_index.png)

  ### データクエリ
  データベース、テーブル、または特定のテーブルを右クリックし、`Query Console`を選択すると、SQLを使って任意の条件のデータを照会することができます。
  ['open_table'](/image/connection-manage/new_console.png)

  「/」を入力してAIを呼び出し、自然言語で入力し、先ほど作成したAIデータセットを選択すると、SQL文が生成されます：

  [aidata](/image/start-guide/nltosql.png)

## 第七步：Chat2DBデータ分析を使用する

### 新しいダッシュボードの作成

左側の`Dashboard`をクリックし、「New Dashboard」をクリックして新しいデータダッシュボードを作成します。レポート名（必須）とレポート説明（任意）を入力します。

['Dashboard'](/image/start-guide/newdashboard.png)

その後、手動でデータレポートを作成することも、またはAIでレポートを作成することもできます。

### グラフの作成

#### 1. AIによるグラフ作成

「AI Chart」をクリックし、右上に表示される入力欄に自然言語で作成したいレポート情報を入力し、先ほど作成したAIデータセットを選択すると、可視化グラフが生成されます：

['Dashboard'](/image/start-guide/aidashboard.png)

['Dashboard'](/image/start-guide/aidashboard2.png)

#### 2. 手動でのグラフ作成

1. 新しいグラフの作成

`New Chart`をクリックします:

['chart'](/image/start-guide/newchart.png)

2. データ設定

上記の図にある`Data configuration`をクリックしてデータ設定を行います。右上から対応するデータベースを選択し、入力欄に適切なSQL文を書き込み、Runをクリックして下部で実行結果を確認します。

['chart'](/image/start-guide/dataconfiguration.png)

その後、`Back to Chart`をクリックしてチャートタイプを選択し、選択されたタイプに基づいて対応する変数（X軸・Y軸座標、パーセンテージなど...）を決定します。

['chart'](/image/start-guide/dataconfiguration2.png)

選択が完了したら、`save`をクリックして保存します！
