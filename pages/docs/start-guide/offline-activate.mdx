# 完全离线模式

<div style={{ fontWeight: 'bold', color: 'red', margin: '20px 0' }}>
注意：每生成一个离线证书需要绑定一个设备，离线证书生成之后无法撤回且不可以修改设备，license可绑定的设备数量减一。
</div>

## 获取设备ID

在激活页面选择离线激活，自动获取设备 ID：

<div style={{maxWidth: '500px'}}>
![获取设备ID](/image/start-guide/device-id.png)
</div>

## 获取设备证书

前往网页端获取设备证书：http://app.chat2db-ai.com/settings/deviceCer

首先登录购买 License 的账号，在 License ID 中选择之前购买的订单，然后依次填入设备名称、选择操作系统，并在最后一栏设备 ID 中填入刚才生成的设备 ID：

![获取设备证书](/image/start-guide/device-cer.png)

点击获取设备证书，即可生成设备证书：

<div style={{maxWidth: '500px' , margin: '20px 0'}}>
![获取设备证书](/image/start-guide/device-cer-dialog.png)
</div>

## 激活设备

在设备证书栏中复制粘贴之前生成的设备证书，即可激活：

<div style={{maxWidth: '500px'}}>
![获取设备ID](/image/start-guide/device-id.png)
</div>

## 如何查看设备证书

前往[网页端](http://app.chat2db-ai.com)登录->设置->购买详情，即可查看之前生成的设备证书。

![设备证书列表](/image/start-guide/device-cer-list.png)

## 如何解绑

如需解绑，请前往客户端，自行移除离线证书。

<div style={{ fontWeight: 'bold', color: 'red', margin: '20px 0' }}>
注意：如果通过离线激活方式完成激活，那么之后即使在联网情况下也无法使用AI功能。如需使用，请先前往客户端后台解除设备证书进行解绑，再通过在线激活进行激活。但是之前生成的证书不会撤回，即激活次数不会恢复。
</div>
