# 教程

## 数据库管理

    1.点击左侧第一个“工作台”按钮，进入数据库管理界面。   
    2.在2、3区域可以切换数据库和schema。  
    3.在4、5区域可查看表结构。  
    4.点击“新建控制台”。在控制台可以对该数据库做操作。  
    !['console'](/image/tutorial-image/manage-database.png)  <br/><br/>
    

    #### 执行SQL
    1、选中SQL，点击执行按钮，可以执行SQL。控制台下方会显示执行结果，可以分页查看结果，也可以导出结果。  
    2、选中SQL，点击格式化按钮，可以格式化SQL。  
    3、点击控制台保存按钮，可以保存SQL到本地，在左侧我的保存可以查看保存记录，下次打开可以直接打开。  
    !['sql'](/image/tutorial-image/manage-database2.png)
