# ライセンスのアクティベートとアンバインド

## ライセンスのアクティベーション

Chat2DB Local Licenseを購入し取得したら、Chat2DB Localクライアントでアクティベーションを行ってください。

ポップアップウィンドウ内の**「アクティベーションコードの確認」**をクリックしてアクティベーションに進んでください。

[アクティベーションコードのポップアップ](/image/start-guide/license1.jpg)

あなたのライセンスを入力してアクティベーションを行ってください：

[アクティベーションコードのポップアップ](/image/start-guide/license2.jpg)

最初はライセンスの確認なしで14日間無料でお試しいただけます。お試し期間中にアクティベーションコードの有効化が必要な場合は、**左上の設定 -> アクティベーションコード**ページへ行き、アクティベーションをクリックしてください：

[アクティベーションコードのポップアップ](/image/start-guide/license3.jpg)

アクティベーションに失敗した場合は、以下のリンクをご覧ください：[Localライセンスのアクティベーション失敗](/docs/question/fail-to-activate-license)

## ライセンスの解除

ライセンスの解除が必要な場合は、**左上の設定 -> アクティベーションコード**ページへ行き、解除をクリックしてください：

[解除](/image/start-guide/unbind.jpg)

以前に紐付けたライセンスを入力して解除を行ってください：

[解除](/image/start-guide/unbind1.jpg)

解除後、ライセンスを他のデバイスに紐付けることができます。手順は上記と同じです。
