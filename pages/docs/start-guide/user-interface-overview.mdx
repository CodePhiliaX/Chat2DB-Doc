# 用户界面概述

Chat2DB 窗口包含**一个菜单栏、一个快捷栏、一个数据库资源管理器（view）、一个工具栏、一个导航栏、一个 "包含一个或多个编辑器和视图" 的工作区以及一个状态栏**：

!["user-interface"](/image/start-guide/user-interface.png)

## Menu Bar 菜单栏

默认情况下，菜单栏包含以下菜单：

- `Chat2DB Pro`菜单包含**重启**、**退出**的菜单项。
- `编辑`菜单包含针对活动元素的全局命令，例如**剪切、复制、粘贴、全选、撤销和重做**操作。
- `视图`菜单包含用于**刷新、放大缩小、缩放视图、全屏**操作的菜单项。
- `帮助`菜单包含**官网、文档**的链接，以及用于检查**日志和控制台**的菜单项。

## Toolbar 工具栏

工具栏包含最基本**创建数据库连接**、**刷新**、**定位**的按钮。

欲了解更多信息，请参阅[工具栏的使用](https://docs.chat2db.ai/docs/connection-manage/Using-toolbar)

## View 数据库资源管理器

数据库资源管理器显示数据库的结构，包括表、视图、存储过程、函数、触发器等，还可以连接各种数据库。

更多信息，请参阅[数据库资源管理相关操作](http://docs.chat2db.ai/docs/connection-manage/data-resource-management)

!["view"](/image/start-guide/view.png)

## Navigation Bar 导航栏

导航栏包含以下项目：

1. **个人信息**：代表个人资料或者账户设置。点击后，您可以查看并编辑您的个人信息，如头像、用户名等，并且可以更改密码或其他相关设置。有关个人信息更多信息，请参阅[个人信息](https://docs.chat2db.ai/docs/settings/personal)

2. **工作空间（Workspace）**：工作空间是您可以与文件和数据库内容进行交互的窗口。

3. **仪表盘（Dashboard）**：仪表盘提供了一个概览视图，展示关键性能指标和其他重要信息。根据您的需求和偏好，您可以选择不同类型的仪表盘，以获取不同形状的图表。有关仪表盘更多信息，请参阅[仪表盘](https://docs.chat2db.ai/docs/dashboard/Common-operations-dashboard)

4. **AI聊天（AI Chat）**：AI聊天可能是指与人工智能助手进行交互的功能。您可以在此处提问SQL、寻求帮助或获取相关信息。

5. **插件（Plugin）**：插件允许您扩展应用程序的功能。通过安装和启用各种插件，您可以根据需求定制应用以满足特定的需求。有关特定插件的更多信息，请参阅[插件](https://docs.chat2db.ai/docs/data-migration/plugin)

6. **全局刷新**：全局刷新按钮用于更新整个页面的内容。当您希望重新加载所有数据时，可以使用此功能。

7. **通知（Notification）**：通知图标显示未读消息的数量。单击该图标可查看来自应用程序的各种通知，例如新功能、提醒等。

8. **反馈（Feedback）**：反馈功能使您能够向开发者发送意见或报告问题。这有助于改进产品和服务质量。有关反馈的更多信息，请参阅[提交反馈](https://docs.chat2db.ai/docs/feedback/give-feedback)

9. **升级会员（Upgrade）**：升级会员按钮用于升级您的会员计划。通过升级会员计划，您可以享受更高的服务级别和功能，例如更多数据库连接、更多存储空间等。有关升级会员更多信息，请参阅[订阅计划](https://docs.chat2db.ai/docs/start-guide/subscription-plan)

## Workspace 工作区

编辑器是您可以与文件和数据库内容进行交互的窗口。工作区中可以同时打开多个视图和编辑器，但一次只能有一个处于活动状态。

表视图和编辑器都可以显示为单独的窗口或选项卡，与选项卡式窗口中的其他视图/编辑器堆叠在一起。
下图显示了选项卡式窗口的标题栏。要查看隐藏选项卡的列表，请单击⬇️图标，可以看到列表并且可以搜索。

!["addmore"](/image/start-guide/addmore.png)

**欲了解更多信息，请参阅[工作空间的使用](http://docs.chat2db.ai/docs/connection-manage/workspace)**


### 查询控制台

新建Console快捷键：
> win: `Ctrl + Shift + L`

> mac: `Command⌘ + Shift + L`

使用查询控制台读取、写入和运行 SQL 代码。

!["console"](/image/start-guide/console01.png)

要关闭视图或编辑器，请单击`关闭`按钮，或右键单击视图/编辑器的标题栏，然后单击上下文菜单上的选项之一（它们根据窗口的配置而变化）：

- 关闭: 关闭活动窗口或选项卡式窗口中的选项卡。
- 关闭其他:（对于在选项卡式窗口中显示为选项卡的编辑器和视图）- 关闭当前选项卡式窗口中除活动选项卡之外的所有选项卡。
- 全部关闭: 关闭选项卡式窗口的所有选项卡（关闭窗口）。

### 数据编辑器和查看器

使用数据编辑器对数据库对象、CSV 文件和查询结果的数据进行排序、筛选、添加、编辑和删除。

!["console"](/image/start-guide/console02.png)

您可以调整表视图窗口的大小。要调整大小，请将光标放在窗口边框上，直到它变为双端箭头，然后单击边缘并将其拖动到所需的大小。

### 查询结果选项卡

执行SQL快捷键：
> win: `Ctrl + R`

> mac: `Command⌘ + R`

在服务工具窗口结果选项卡中查看查询的结果集。

!["console"](/image/start-guide/console03.png)

## Status Bar 状态栏

主窗口底部状态栏的右侧部分显示最近的导出进度。单击快捷栏中的`导出进度`，并点击`Show all`可将其打开。

!["exportProgress"](/image/start-guide/exportProgress.png)
