# サブスクリプションプラン

## 購読プロセス

> 会員プログラムの詳細については、[会員規約](https://docs.chat2db.ai/ja-JP/docs/service/member)をご覧ください。

まず、Chat2DB アカウントにログインする必要があります。

プランにサブスクライブすると、サブスクリプション期間中、Chat2DB アカウントにログインして Chat2DB Pro を使用できます。

**アバターをクリック**するか、**個人情報を表示**して、サブスクリプションの有効期限を確認できます。メンバーシップの有効期限が切れた後は、プランを再度更新できます。

!["計画"](/image/start-guide/plan01.png)

サブスクリプション プランは 2 つのカテゴリに分類されます。

- **個人版メンバーシップ**: 個人の開発者、データ アナリスト、または小規模なプロジェクト チームに適しています。割引コードを入力すると割引を受けられます。

!["計画"](/image/start-guide/plan02.png)


- **チーム エディション メンバー**: 中規模および大規模なチームまたはエンタープライズ レベルのユーザーに適しています。割引コードを入力すると割引を受けられます。

!["計画"](/image/start-guide/plan03.png)

**サブスクリプションが成功した後にシステムが応答しない場合は、アプリケーションを再起動してください。**
