# Local 授权码激活/解绑

## 授权码激活

购买并获得 Chat2DB Local License 后，请前往 Chat2DB Local 客户端进行激活。

点击弹窗中的 **"验证激活码"** 前往激活。

![激活码弹窗](/image/start-guide/license1.jpg)

输入你的 License 进行激活：

![激活码弹窗](/image/start-guide/license2.jpg)

首次可以免费试用 14 天，无需验证 License。如果你已处于试用期，并且需要激活验证码，请前往**左上角设置 -> 激活码**页面，点击激活：

![激活码弹窗](/image/start-guide/license3.jpg)

如果激活失败，请查阅：[Local 授权码激活失败](/docs/question/fail-to-activate-license)

## 授权码解绑

### 1. 前往客户端解绑

如果需要解绑授权码，可以前往**左上角设置->激活码**页面，点击解绑：

![解绑](/image/start-guide/unbind.jpg)

填写之前绑定的 License 进行解绑：

![解绑](/image/start-guide/unbind1.jpg)

解绑后可以将 License 绑定在其他设备上，操作步骤同上。

### 2. 网页端解绑

如果你的绑定设备无法继续使用，可使用任何设备前往网页端：https://app.chat2db-ai.com/workspace自行解绑。

依次点击**左上角头像** -> **设置** -> **购买详情**

![解绑](/image/start-guide/license4.png)
