# インストールガイド

開いているアプリケーションをすべて閉じることを強くお勧めします。これにより、スムーズな取り付けが可能になります。

Chat2DB は次のオペレーティング システムをサポートしています。

- Windows: Windows x64。
- MacOS: macOS Intel/Apple M チップ。
- Linux: .AppImage としてパッケージ化され、x86_64 および arm64 をサポートします。

## Windows

1. Windows バージョンの Chat2DB の `.exe` インストーラーをダウンロードします。

2. 「.exe」ファイルをダブルクリックして、インストール ウィザードを開始します。

3. インストール ウィザードの指示に従って、インストール プロセスを完了します。


## MacOS

1. Chat2DB macOS 版をダウンロードします。

2. `.dmg` ファイルを開きます。

3. Chat2DB をアプリケーション フォルダーにドラッグしてインストールします。


## Linux

1. 対応するダウンロード リンク (`.ApplImage x86` または `.ApplImage arm64`) をクリックして、アプリケーション ファイルをダウンロードします。

2. 端末内で、ダウンロードしたアプリケーション ファイルが含まれるディレクトリに移動します。

3. アプリケーション ファイルに実行権限を追加します。

```bash
chmod +x <downloaded_file>
```
ここで、`<downloaded_file>` はダウンロードしたアプリケーションのファイル名です (例: `chat2db-linux-x86.applage`)。

4. アプリケーションを実行します。

```bash
./<downloaded_file>
```

これでアプリケーションが実行されるはずです。 Linux ディストリビューションが非標準リポジトリからのアプリケーションのインストールを許可していることを確認してください。そうでない場合は、そのようなアプリケーションの実行を許可するようにセキュリティ設定を調整する必要があるかもしれません。

## 自動更新

この機能は「Windows」と「macOS」でのみ利用できます。

Chat2DB は、新しいバージョンを定期的に自動的にチェックします。チェックが無効になっておらず、新しいバージョンが利用可能な場合は、アップグレードするかどうかを決定するよう求められます。アップグレードを選択した場合、インストーラーが自動的にダウンロードされ、完了時に起動します。

デフォルトでは、Chat2DB は更新を自動的にチェックし、新しいバージョンが利用可能になったときに通知するように構成されています。
通常、更新はパッチベースです。更新は既存のインストールに適用され、アプリを再起動するだけで済みます。
ただし、パッチ更新が利用できない場合があり、新しいバージョンの Chat2DB を [公式 Web サイト](https://chat2db-ai.com/download) にインストールする必要があります。

Chat2DB がローカル ネットワークの外部に HTTP アクセスできない場合、更新を確認してパッチを適用することができません。
この場合、Chat2DB の新しいバージョンをダウンロードして手動でインストールする必要があります。

> 注: Chat2DB 自体は、データの損失を避けるために正常にシャットダウンします。
