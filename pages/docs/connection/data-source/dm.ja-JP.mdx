# DMに接続

サーバー オブジェクトの使用を開始するには、接続を作成して開く必要があります。サーバーが空の場合は、新しいデータベースを作成する必要があります。

Chat2DB を使用して DM サーバーにすばやく接続します。

## 公式ドキュメントとソフトウェア:

- DM の詳細については、[公式ドキュメント](https://eco.dameng.com/document/dm/zh-cn/pm/index.html) を参照してください。
- DMデータベースソフトウェアのダウンロードについては、[公式ソフトウェアダウンロード](https://www.dameng.com/list_103.html)を参照してください。

## ステップ 1: 新しい接続を作成する

**ツールバー**で**新規**アイコン(+)をクリックし、**データソース**に移動して**DM**を選択します。

!['newConnection-接続'](/image/connection/dm1.png)

## ステップ 2: 接続情報を入力します

!['newConnection-接続'](/image/connection/dm2.png)

- `Name`：接続名をカスタマイズすると覚えやすくなります
- `Env`：接続環境、テスト、本番などを選択して、異なる環境での接続を区別します
- `Storge`: 接続ストレージのタイプを選択します。現在はローカル (LOCAL) とクラウド (CLOUD) をサポートしています。
- `Host`: DM サーバー アドレス。IP またはドメイン名にすることができます。
- `Port`: DM サーバー ポート、デフォルトは 5236
- `Authentication`: DM 認証方法、ユーザー名とパスワード、または認証は不要です
- `User`: DM ユーザー名
- `Password`: DM パスワード
- `Database`: DM データベース名、オプション。入力しない場合は、デフォルトのデータベースに接続します。
- `URL`: DM 接続 URL、オプション、入力しない場合は、上記の情報に基づいて自動的に生成されます
- `Driver`: DM ドライバー (オプション)。入力しない場合は、URL に基づいて自動的に識別されます。手動で選択することもできます。手動で選択すると、ドライバー設定項目が表示されます。
- `SSH`: SSH 接続を使用するかどうか、オプション。未入力の場合、デフォルトで無効になります。選択すると、SSH 設定項目が表示されます。
- `Advanced Configuration`: 詳細設定、オプション、未入力の場合はデフォルトでは使用されません。選択後に詳細設定項目が表示されます。

## ステップ 3: ドライバーをダウンロードする

DM の JDBC ドライバーをダウンロードしたい場合は、ダイアログ ボックスの下部にある「ドライバーのダウンロード」ダウンロード ボタンをクリックしてください。もちろん、独自のドライバーを手動でアップロードすることもできます (「ドライバーのアップロード」)。

!['ドライバー'](/image/connection/driver.png)

## ステップ 4: SSH チャネルを構成する (オプション)

!['ssh'](/image/connection/ssh.png)
- `USE SSH`：SSHトンネルを使用するかどうか
- `SSH Host`：SSHサーバーアドレス
- `SSH Port`：SSHサーバーポートのデフォルトは22
- `Authentication`：SSH 認証方法、ユーザー名、パスワード、または秘密キー
- `User`：SSHユーザー名
- `Password`：SSHパスワード
- `Test the SSH connection`：SSH接続が正常かどうかをテストする

## ステップ 5: 接続をテストする

提供された詳細を使用してデータベースへの接続を確立できることを確認します。
これを行うには、左下隅にあるテスト ボタンをクリックして、接続が正常であるかどうかをテストします。正常でない場合は、接続に失敗します。エラーメッセージ。

!['ssh'](/image/connection/test.png)

**接続に関する問題が発生した場合は、[データベースに接続できない](https://docs.chat2db.ai/ja-JP/docs/question/Cannot-connect-to-database)ページを参照してください。**

## ステップ 6: 接続を保存する

テスト接続が成功したら、「保存」ボタンをクリックして接続情報を保存します。接続情報をクリックして接続の詳細を表示するか、接続を削除します。

!['ssh'](/image/connection/test.png)
