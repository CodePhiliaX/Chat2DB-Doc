# 连接 Redis

要开始使用服务器对象，您应该创建并打开一个连接。如果服务器是空的，则需要创建一个新的数据库。

用 Chat2DB 快速连接 Redis 服务器。

## 官方文档和软件：

- 有关 Redis 的完整信息，请参阅[官方文档](https://redis.io/docs/latest/)
- 下载 Redis 数据库软件，请参考[官方软件下载](https://redis.io/downloads/)

## 第一步：新建连接

在**工具栏**中，单击**新建**图标 (+)，导航到**数据源**并选择 **Redis**。

!['redis2'](/image/connection/redis2.png)

## 第二步：填写连接信息

!['Redis'](/image/connection/redis.png)

- `Name`：连接名称自定义可以方便记忆
- `Env`：选择连接环境，测试、生产等用于区分不同环境的连接
- `Storge`：选择连接存储类型，目前支持本地(LOCAL)、云端(CLOUD)
- `Host`：Redis 服务器地址，可以是 IP 或者域名
- `Port`：Redis 服务器端口，默认 6379
- `Authentication`：Redis 认证方式，用户名密码或者不需要认证
- `User`：Redis 用户名
- `Password`：Redis 密码
- `Database`：Redis 数据库名称，选填，不填会连接到默认数据库
- `URL`：Redis 连接 URL，选填，不填会根据上面的信息自动生成
- `Driver`：Redis 驱动，选填，不填会根据 URL 自动识别, 也可以手动选择，手动选择后会显示驱动配置项
- `SSH`：是否使用 SSH 连接，选填，不填默认不使用, 选择后会显示 SSH 配置项
- `Advanced Configuration`：高级配置，选填，不填默认不使用, 选择后会显示高级配置项

## 第三步：下载驱动

如果要下载 Redis 的 JDBC 驱动程序，请单击对话框底部的`Download Driver`下载按钮，当然也可以手动上传自己的驱动（`Upload driver`）

!['driver'](/image/connection/driver.png)

## 第四步：配置 SSH 通道(可选)

!['ssh'](/image/connection/ssh.png)
- `USE SSH`：是否使用 SSH 通道
- `SSH Host`：SSH 服务器地址
- `SSH Port`：SSH 服务器端口 默认 22
- `Authentication`：SSH 认证方式，用户名密码或者私钥
- `User`：SSH 用户名
- `Password`：SSH 密码
- `Test the SSH connection`：测试 SSH 连接是否正常

## 第五步：测试连接

确保可以使用提供的详细信息建立与数据库的连接。
为此，点击左下角的测试按钮，测试连接是否正常，如果连接正常会显示连接成功，否则会显示连接失败，可以根据错误信息调整连接信息。

!['ssh'](/image/connection/test.png)

**如果出现任何连接问题，请参阅[无法连接到数据库](https://docs.chat2db.ai/docs/question/Cannot-connect-to-database)页面。**

## 第六步：保存连接

测试连接通过后，点击保存按钮，保存连接信息，连接信息会显示在左侧的数据库列表中，点击连接信息可以查看连接详情，也可以删除连接。

!['ssh'](/image/connection/test.png)
