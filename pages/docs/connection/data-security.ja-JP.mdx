# データセキュリティ

データはビジネスの中核資産であるため、データ セキュリティはビジネスの重要な要素です。データ セキュリティの確保は **Chat2DB** の中核的な目標です。

この目標を達成するために、**Chat2DB** はデータのセキュリティを確保するために次の措置を講じています。

## データストレージ

- **Chat2DB** は、不正なデータ アクセスを防ぐための高度な物理的および技術的セキュリティ対策を備えた、厳密にセキュリティ認定されたデータ センターにデータを保存します。

- ユーザーの機密情報と重要な機密データについては、**Chat2DB** は非対称暗号化ストレージ スキームを実装し、プラットフォーム内部関係者であってもこれらのデータに直接アクセスできないようにします。

##送信暗号化

Chat2DB は、**SSL/TLS プロトコル**を使用してデータ送信を暗号化します。これは、送信中のデータのセキュリティと整合性を効果的に保護できる業界標準の安全な通信プロトコルです。

## データアクセス

データのセキュリティを最大限に高めるために、Chat2DB クライアントをインストールして使用することをお勧めします。
クライアントがユーザーのデータベース サーバーに直接接続することにより、データ送信プロセス全体が Chat2DB プラットフォームを経由する必要がなくなり、潜在的なデータ漏洩のリスクが軽減されます。

## プライベート展開

Chat2DB はプライベート展開オプションをサポートしており、企業が自社のインフラストラクチャにアプリケーションを展開できるようになり、データの制御とセキュリティがさらに強化されます。
