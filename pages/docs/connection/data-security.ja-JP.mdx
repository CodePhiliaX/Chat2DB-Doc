# データセキュリティ

データは企業の中核資産であるため、データ セキュリティは企業にとって非常に重要な部分です。 データのセキュリティをどのように確保するかは、Chat2DB の重要な目標です。

Chat2DB は、次の方法でデータのセキュリティを確保します。

## データストレージ

Chat2DB は、データが盗まれないように厳格なセキュリティ対策を講じた安全なデータセンターにデータを保存します。 同時に、Chat2DB はユーザーの重要な機密データの非対称暗号化と保存を実行し、プラットフォームはユーザーの重要な機密データにアクセスできなくなります。

## 送信暗号化

Chat2DB は、SSL/TLS 暗号化を通じて送信中のデータのセキュリティを保証します。 SSL/TLS は、送信中にデータが盗まれないようにする暗号化プロトコルです。

## データアクセス
データのセキュリティを最大限に確保するために、Chat2DB クライアントをインストールし、クライアントを通じてデータベースにアクセスすることをお勧めします。ローカル クライアントは、Chat2DB プラットフォームを経由せずにユーザーのデータベース サーバーに直接接続します。

## プライベート展開

Chat2DB はプライベート展開をサポートしており、企業は Chat2DB を自社のサーバーに展開できるため、データ セキュリティを最大限に確保できます。
