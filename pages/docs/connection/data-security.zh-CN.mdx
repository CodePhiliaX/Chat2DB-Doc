# 数据安全

数据安全是企业至关重要的组成部分，因为数据是企业的核心资产。确保数据的安全性是 **Chat2DB** 的一个核心目标。

为了达成这一目标，**Chat2DB** 采取了以下措施来保障数据的安全：

## 数据存储

- **Chat2DB** 将数据存储在经过严格安全认证的数据中心内，这些数据中心配备有先进的物理和技术安全措施，以防止未经授权的数据访问。

- 对于用户的敏感信息和核心机密数据，**Chat2DB** 实施非对称加密存储方案，确保即使平台内部人员也无法直接访问这些数据。

## 传输加密

Chat2DB 采用 **SSL/TLS 协议**对数据传输进行加密处理，这是一种行业标准的安全通信协议，可有效保护数据在传输过程中的安全性和完整性。

## 数据访问

为了最大限度地保障数据安全，我们建议用户安装并使用 Chat2DB 客户端。
通过客户端直接连接到用户的数据库服务器，数据传输全程无需经过 Chat2DB 平台，从而减少潜在的数据泄露风险。

## 私有化部署

Chat2DB 支持私有化部署选项，允许企业在自己的基础设施上部署该应用，进一步增强对数据的控制和安全性。

私有化部署详情：[https://github.com/chat2db/Chat2DB/wiki](https://github.com/chat2db/Chat2DB/wiki)
