# エクスポートとインポート

いくつかの方法を使用して、データベースからデータとオブジェクト構造をエクスポートできます。さらに、データを SQL、CSV、XLS、XLSX 形式にエクスポートできます。
Chat2DB では、オブジェクト構造とデータを個別にエクスポートできます。これは、テーブルまたはビューの構造をエクスポートしてから、これらのオブジェクトからデータをエクスポートできることを意味します。

[こちら](https://docs.chat2db.ai/ja-JP/docs/start-guide/user-interface-overview#status-bar-%E7%8A%B6%E6%80%81%E6%A0%8F) 、エクスポートの進行状況を確認します。


## オブジェクト構造をエクスポートする

> ヒント: データの整合性に関する潜在的な問題を防ぐため、エクスポート プロセス中にデータを変更しないことをお勧めします。

### データベース全体をエクスポートする

1. オブジェクト構造をエクスポートするデータベースを選択し、「構造のみをエクスポート」をクリックします。

!["エクスポート"](/image/connection-manage/export01.png)

2. 保存パスを選択した後、「ファイルを開く」をクリックして、エクスポートが完了した後にエクスポートされたファイルを表示します。

!["エクスポート"](/image/connection-manage/export02.png)

3. エクスポートが成功しました。

!["エクスポート"](/image/connection-manage/export03.png)


### 単一のテーブルをエクスポートする

方法は上記と同じで、**単一テーブル**を選択して操作するだけです。

## オブジェクトの構造とデータをエクスポートする

> ヒント: データの整合性に関する潜在的な問題を防ぐため、エクスポート プロセス中にデータを変更しないことをお勧めします。

### データベース全体をエクスポートする

1. オブジェクト構造をエクスポートするデータベースを選択し、「構造とデータのエクスポート」をクリックします。

!["エクスポート"](/image/connection-manage/export07.png)

2. 保存パスを選択した後、「ファイルを開く」をクリックして、エクスポートが完了した後にエクスポートされたファイルを表示します。

!["エクスポート"](/image/connection-manage/export08.png)

3. エクスポートが成功しました。

!["エクスポート"](/image/connection-manage/export09.png)

### 単一のテーブルをエクスポートする

合計 3 つの方法があります。最初の方法は上記と同じで、操作する **単一テーブル** を選択するだけです。

**2 番目の方法**:
「データのエクスポート」を選択し、エクスポートするファイル形式と保存場所を選択します。

!["エクスポート"](/image/connection-manage/export04.png)

!["エクスポート"](/image/connection-manage/export05.png)

!["エクスポート"](/image/connection-manage/export06.png)

**3 番目の方法**:
エディター バーの右側にある [エクスポート] をクリックしてデータをエクスポートします。

!['export1'](/image/connection-manage/export1.png)

- `Export result set csv`: クエリ結果セット全体を CSV ファイルにエクスポートします。
- `Export result set insert sql`: データベース クエリの結果を一連の SQL INSERT ステートメントとしてエクスポートします。
- `現在のページセットの結果をcsvでエクスポート`: 現在のページの結果をCSV形式のファイルにエクスポートします。
- `Export result of current page set insert sql`: 現在のページに表示されている結果を INSERT SQL ステートメントとしてエクスポートします。

## オブジェクトの構造とデータをインポートする

### データベース全体をインポートする

ライブラリをランダムに選択し、右クリックして「SQL ファイルの実行」を選択してインポートします。

!["エクスポート"](/image/connection-manage/import001.png)


### 単一のテーブルをインポートする

1. オブジェクト構造をインポートするテーブルを選択し、「データのインポート」をクリックします。

!["エクスポート"](/image/connection-manage/import01.png)

2. インポートするファイルの形式と保存パスを選択します。

!["エクスポート"](/image/connection-manage/import02.png)

3. インポートが成功しました。

!["エクスポート"](/image/connection-manage/import03.png)
