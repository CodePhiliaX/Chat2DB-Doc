# 数据导出

在工具栏右侧点击`Export`进行数据导出：

!['export1'](/image/connection-manage/export1.png)

- `Export result set csv`：将整个查询结果集导出为CSV文件
- `Export result set insert sql`：将数据库查询结果导出为一系列的SQL INSERT语句
- `Export result of current page set csv`：将当前页面上的结果导出为CSV格式的文件
- `Export result of current page set insert sql`：将当前页面显示的结果导出为INSERT SQL语句
