# 导出和导入

您可以使用多种方法从数据库导出数据和对象结构。此外，您还可以导出为 SQL、CSV、XLS、XLSX 格式的数据。
在 Chat2DB 中，您可以分别导出对象结构和数据。这意味着您可以导出表或视图的结构，然后从这些对象中导出数据。

[在这里](https://docs.chat2db.ai/docs/start-guide/user-interface-overview#status-bar-%E7%8A%B6%E6%80%81%E6%A0%8F)，查看导出进度。


## 导出对象结构

> 提示：建议在导出过程中不要更改数据，以防止出现任何潜在的数据完整性问题。

### 导出整个数据库

1. 选中你要导出对象结构的数据库，点击`Export structure only`。

!["export"](/image/connection-manage/export01.png)

2. 选定好保存路径后，待导出完成点击`Open file`查看导出文件。

!["export"](/image/connection-manage/export02.png)

3. 导出成功！

!["export"](/image/connection-manage/export03.png)


### 导出单个表

方法同上，选择**单个表**进行操作即可。

## 导出对象结构和数据

> 提示：建议在导出过程中不要更改数据，以防止出现任何潜在的数据完整性问题。

### 导出整个数据库

1. 选中你要导出对象结构的数据库，点击`Export structure and data`。

!["export"](/image/connection-manage/export07.png)

2. 选定好保存路径后，待导出完成点击`Open file`查看导出文件。

!["export"](/image/connection-manage/export08.png)

3. 导出成功！

!["export"](/image/connection-manage/export09.png)

### 导出单个表

一共三种方法：第一种方法同上，选择**单个表**进行操作即可。

**第二种方法**：
选择`Export data`，并选择你要导出的文件格式及保存位置。

!["export"](/image/connection-manage/export04.png)

!["export"](/image/connection-manage/export05.png)

!["export"](/image/connection-manage/export06.png)

**第三种方法**：
在编辑器栏右侧点击`Export`进行数据导出：

!['export1'](/image/connection-manage/export1.png)

- `Export result set csv`：将整个查询结果集导出为CSV文件
- `Export result set insert sql`：将数据库查询结果导出为一系列的SQL INSERT语句
- `Export result of current page set csv`：将当前页面上的结果导出为CSV格式的文件
- `Export result of current page set insert sql`：将当前页面显示的结果导出为INSERT SQL语句

## 导入对象结构和数据

### 导入整个数据库

随机选择一个库，鼠标右键选择`Run SQL file`即可导入！

!["export"](/image/connection-manage/import001.png)


### 导入单个表

1. 选中你要导入对象结构的表，点击`import data`。

!["export"](/image/connection-manage/import01.png)

2. 选好要导入文件的格式和保存路径。

!["export"](/image/connection-manage/import02.png)

3. 导入成功！

!["export"](/image/connection-manage/import03.png)
