# ツールバーの使い方

!['データベース管理'](/image/connection-manage/databasemanage77.png)

---

## 1. 左上隅の「+」をクリックして、新しいデータベースを作成します。

!['データベース管理'](/image/connection-manage/databasemanage55.png)

- `新しいグループ`: 新しいグループを作成します。

!['データベース管理'](/image/connection-manage/databasemanage44.png)

- 「新しい接続」: 新しい接続を作成します。

!['データベース管理'](/image/connection-manage/databasemanage22.png)

- `接続のインポート`: 接続のインポート

!['データベース管理'](/image/connection-manage/databasemanage33.png)



## 2. 左上隅の「更新」ボタンをクリックして、データベースのリストを更新します。

## 3. 左上隅の「選択したノードまでスクロール」をクリックして、選択したノードまでスクロールします。

!['データベース管理'](/image/connection-manage/Scroll.png)

## 4. 上の [列コメントを表示] をクリックして列コメントを表示します。デフォルトでは、フィールド タイプが表示されます。

!['データベース管理'](/image/connection-manage/column-comment2.png)

!['データベース管理'](/image/connection-manage/column-comment3.png)

## 5. 右側の検索ボックスをクリックして、主要なテーブル ライブラリをすばやく取得します。

!['データベース管理'](/image/connection-manage/searchBar.png)
