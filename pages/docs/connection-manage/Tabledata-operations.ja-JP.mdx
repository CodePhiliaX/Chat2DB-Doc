# テーブルデータに対する関連操作

## 行と行の操作

!['operations'](/image/connection-manage/operations_data2.png)
データ行を選択するときにマウスを右クリックすると、次の操作が実行されます。

- `Insert SQL`: データ行全体を Insert SQL ステートメントにコピーします。

!['Insert SQL'](/image/connection-manage/Insert_SQL.png)

- `Update SQL`: データ行全体を Update SQL ステートメントにコピーします。

!['Update SQL'](/image/connection-manage/Update_SQL.png)

- `TAB delimited (data)`: データ行全体をコピーします。

!['TAB delimited'](/image/connection-manage/TAB_data.png)

- `TAB delimited (field name)`: データ行全体に対応するフィールド名をコピーします。

!['TAB delimited'](/image/connection-manage/TAB_field.png)

- `Tab-separated (field names and data)`: データの行全体とそれに対応するフィールド名をコピーします。

!['TAB delimited'](/image/connection-manage/TAB_all.png)

- `Clone row`: データ行全体のクローンを作成します。

- `Delete row`: データの行全体を削除します。

**チップ**：
- 複数行のデータをコピーする場合は、「Shift + マウスの左ボタン」を使用して複数の行を選択して操作してください。
- テーブル内のデータの行全体をコピーする場合は、マウスの左上隅にある空白領域を左クリックして、すべてを選択します (以下を参照)。

!['select all'](/image/connection-manage/selectAll.png)

## テーブルデータの操作

!['operations2'](/image/connection-manage/operations_data.png)

- `View/Edit Data`: データの表示/編集
- `Copy`: 選択した単一データをコピーします
- `Copy the row as`: 上と同じ...
- `Clone row`: 行データのクローンを作成します。
- `Set NULL`: テーブルデータを **null** に設定します
- `Set DEFAULT`: テーブル データを **デフォルト値** に設定します
- `Delete row`: 行を削除します。

## 指定されたデータをクエリする
!['designated'](/image/connection-manage/designated.png)

条件付きフィルタリングと並べ替えをテーブルヘッダーで直接実行し、`Enter`をクリックして結果を取得できます。


## テーブルデータのソート

!['sort'](/image/connection-manage/sort_data.png)

テーブルのフィールド名の横にある小さな三角形をクリックすると、フィールド属性に基づいて昇順または降順ですばやく並べ替えられます。


**注意**: 並べ替えるには 1 つの列のみを選択できます。複数の列を選択して並べ替えることはできません。
