# Using workspace

## Query designated data

!['designated'](/image/connection-manage/designated.png)

You can directly filter and sort conditions in the table header, click `Enter` to get the results

## Sort table data

!['sort'](/image/connection-manage/sort_data.png)

Click the small triangle next to the table field name to quickly sort in ascending or descending order according to the field attributes.

**Note**: You can only select one column to sort it, and you cannot select multiple columns to sort

## Save Consoles

Click `Save` in the toolbar to save all the contents written in the console (even if the SQL statement is not executed)!

!['save'](/image/connection-manage/save.png)

## Format SQL

Click `Format` in the toolbar to format the SQL statement!

!['format'](/image/connection-manage/format.png)

## Save SQL locally

Click `Save as file` in the toolbar to save SQL statements locally!

!['save'](/image/connection-manage/saveAsFile.png)
