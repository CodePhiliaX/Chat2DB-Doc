# 数据库管理

打开链接后，可以看到数据库管理界面，可以对数据库进行增删改查操作。

## 切换链接

点击右上角的 `切换链接` 按钮，可以切换链接。
!['changeconnection'](/image/connection-manage/changeconnection.png)


## 查看管理 database
!['databasemanage'](/image/connection-manage/databasemanage.png)

- 选中链接后，可以看到数据库列表，点击数据库名，可以查看数据库详情。
- 点击左上角“+”可以新建数据库，点击左上角“刷新”按钮可以刷新数据库列表。
- 在 databse 上右键可以刷新、copy等操作。

## 查看管理 table
!['tables'](/image/connection-manage/tables.png)
tables 界面可以查看数据库中的表，可以新建、刷新、查看所有表，也可以刷新所有的表。

## 新建编辑表
!['tables'](/image/connection-manage/create_t_base.png)
新建表填写基础信息
- Table Name: 表名
- Comment: 表注释
- Character set: 字符集
- Engine: 存储引擎
- Increment value: 自增长开始值

!['tables'](/image/connection-manage/create_t_column.png)
填写列信息
- Column Name: 列名
- Type: 类型
- Size: 长度
- Nullable: 是否为空
- Key: 主键
- Comment: 注释

!['tables'](/image/connection-manage/create_t_index.png)
- Index Name: 索引名
- Column: 列名
- Type: 索引类型
- Comment: 注释

## 查看管理 view
!['views'](/image/connection-manage/views.png)
点击左侧菜单views，可以查看数据库中的视图，可以新建、刷新、查看所有视图，也可以刷新所有的视图。
双击可以查看视觉详情。

## 查看管理 index
!['indexs'](/image/connection-manage/indexs.png)
点击左侧菜单indexs，可以查看数据库中的索引，可以新建、刷新、查看所有索引，也可以刷新所有的索引。

## 查看管理 trigger
!['triggers'](/image/connection-manage/triggers.png)
点击左侧菜单triggers，可以查看数据库中的触发器，可以新建、刷新、查看所有触发器，也可以刷新所有的触发器。

## 查看管理 function
!['functions'](/image/connection-manage/functions.png)
点击左侧菜单functions，可以查看数据库中的函数，可以新建、刷新、查看所有函数，也可以刷新所有的函数。
