# 数据库管理

打开链接后，可以看到数据库管理界面，可以对数据库进行增删改查操作。

## 切换链接

点击右上角的 `切换链接` 按钮，可以切换链接。
!['changeconnection'](/image/connection-manage/changeconnection.png)


## 查看管理 database
!['databasemanage'](/image/connection-manage/databasemanage.png)

- 选中链接后，可以看到数据库列表，点击数据库名，可以查看数据库详情。
- 点击左上角“+”可以新建数据库，点击左上角“刷新”按钮可以刷新数据库列表。
- 在 databse 上右键可以刷新、copy等操作。

## 查看管理 table
!['tables'](/image/connection-manage/tables.png)
tables 界面可以查看数据库中的表，可以新建、刷新、查看所有表，也可以刷新所有的表。
